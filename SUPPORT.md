# Support

## Where to Get Help

- Usage questions: open a GitHub Discussion in the project repository.
- Bug reports: open a GitHub Issue using the bug report template.
- Security issues: use the private reporting path in SECURITY.md.

## What to Include

When requesting help, include:

- Package version
- Laravel version
- PHP version
- Database driver/version
- Minimal reproduction steps
- Relevant error output and stack trace

## Before Opening an Issue

- Check README.md and docs/PERFORMANCE_TUNING.md.
- Search existing issues/discussions to avoid duplicates.
